# Food Delivery Web Application

This is a full-stack food delivery web application where users can browse food items, register/login, and place orders.

## Features
- User Authentication (Login/Signup)
- Food Listing
- Cart Functionality
- Order Management

## Tech Stack
- Frontend: React.js
- Backend: Node.js, Express.js
- Database: MongoDB (assumed)

## Description
The project demonstrates how frontend and backend interact using APIs. It focuses on user experience, dynamic content rendering, and basic e-commerce functionality.
